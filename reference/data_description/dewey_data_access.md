# Dewey Data Access Options

Source: https://docs.deweydata.io/docs/data-access-options

## Overview

Dewey provides multiple methods for accessing and consuming data, each suited to different use cases and data sizes.

## Access Methods

### 1. CSV Download
- **Best For**: Smaller datasets
- **Size Limit**: Under 2.0 GB
- **Access**: Directly within Dewey platform and projects
- **Ease of Use**: Simple point-and-click download

### 2. Dewey Client (Recommended)
- **Description**: Dewey's newest and most efficient data download method
- **Best For**: Most use cases
- **Features**:
  - Optimized for performance
  - Detailed guides and packages available
  - Platform's preferred method
- **Status**: Actively maintained and recommended for new users

### 3. Bulk API
- **Description**: Python package for bulk dataset access
- **Status**: Deprecated but still functional
- **Best For**: Legacy workflows, programmatic access
- **Note**: Consider migrating to Dewey Client

## Recommendations

- **New Users**: Start with Dewey Client
- **Small Datasets (<2GB)**: CSV Download is fastest
- **Programmatic Access**: Dewey Client or Bulk API
- **Legacy Systems**: Bulk API still supported

## Getting Started

1. Sign up for Dewey account at deweydata.io
2. Browse available datasets
3. Choose access method based on data size and workflow
4. Follow dataset-specific documentation for schema details
